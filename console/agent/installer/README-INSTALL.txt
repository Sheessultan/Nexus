Console Agent - install on any Windows 10/11 PC
================================================

1) Run ConsoleAgent-Setup.exe AS ADMINISTRATOR
   OR extract ZIP and right-click INSTALL.bat -> Run as administrator

2) Open portal in browser:
   http://52.62.136.167:3000

3) Agent connects to API:
   http://52.62.136.167:4000

Change API URL later:
   Edit C:\Program Files\ConsoleAgent\console-agent.json
   Restart task "ConsoleAgent" in Task Scheduler

Uninstall: Settings -> Apps -> Console Agent
